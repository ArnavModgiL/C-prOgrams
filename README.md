# C-prOgrams :-
This repository contains basic to intermediate C programming programs.
These programs are useful for beginners who are learning C language and want to
understand important programming concepts.

## Topics Covered

- Basic Input / Output Programs
- Conditional Statements
- Loops
- Functions
- Arrays
- Strings
- Recursion
- Pointers
- Pattern Programs
- Number Based Programs

## Programs Included

Some of the programs in this repository include:

- Compare Two Strings Without strcmp()
- Palindrome Number
- Prime Number Check
- Armstrong Number
- String Length Without strlen()
- Factorial Using Recursion
- Power Function Using Recursion
- Pattern Printing Programs

## Purpose of This Repository

The goal of this repository is to practice and improve problem solving skills
in C programming.
